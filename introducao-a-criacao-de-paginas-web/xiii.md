# Xiii... não sei por onde começar...

### Passo 1

Instale o VS-Code a partir do site oficial: [https://code.visualstudio.com/](https://code.visualstudio.com/)

### Passo 2

Crie uma pasta para abrigar os seus projetos HTML

Vamos começar abrindo o VS-Code.
Você deve encontrar uma janela parecida com essa:

![VS-Code inicial](imagens/Xiii-2.png)

Clique no ícone do Explorador no menu alto do menu lateral esquerdo como mostrado a seguir:

![Explorer](imagens/Xiii-3.png)

Como esse é o seu primeiro projeto, não há nada aberto pronto para continuar o desenvolvimento.
Nesse caso, você deve clicar no botão azul `Abrir Pasta`:

![Abrir Pasta](imagens/Xiii-4.png)

A janela `Abrir Pasta` do Windows deverá abrir.
Navegue até o local onde você gostaria de armazenar os seus sites.
No meu caso, eu vou salvar os meus sites em uma pasta dentro de `Meus Documentos`.
Então selecione `Meus Documentos`:

![Meus Documentos](imagens/Xiii-5.png)

Clique no botão `Nova pasta`:

![Nova pasta](imagens/Xiii-6.png)

Na pasta criada, escolha um nome para a pasta que vai abrigar todos os seus sites (pelo menos todos os seus sites que vamos criar nessa disciplina).

![Nova pasta](imagens/Xiii-7.png)

Eu usei, nesse exemplo, o nome `meus-sites` como nome da pasta que vou usar para criar esse exemplo:

![meus-sites](imagens/Xiii-8.png)

Depois de digitar o nome, clique no botão `Selecionar pasta`:

![Selecionar pasta](imagens/Xiii-9.png)

A nova pasta agora está na sua área de trabalho do seu VS-Code:

![Área de Trabalho](imagens/Xiii-10.png)

## Passo 3

Cria uma pasta para abrigar o seu site em desenvolvimento.

Leve o seu mouse até o nome da pasta base (`meus-sites`) e clique no botão `Nova Pasta` para criar uma pasta para o site que vamos criar agora.
Você pode usar esse processo para criar outras pastas para outros sites que você criar posteriormente.

![Nova Pasta](imagens/Xiii-11.png)

Informe um nome para essa nova pasta.
No meu exemplo, eu chamei a pasta que vai conter o site desse exemplo de `exemplo`:

![exemplo](imagens/Xiii-12.png)

## Passo 4

Vamos agora criar o nosso arquivo HTML.

Clique com o botão da direita do mouse no nome da pasta `exemplo` e selecione a opção `Novo arquivo...`:

![Novo Arquivo...](imagens/Xiii-13.png)

Escolha um nome para esse novo arquivo.
Eu escolhi o nome `index.html`:

![index.html](imagens/Xiii-14.png)

Na área de texto localizada na parte direita do VS-Code, coloque o seu texto HTML.
Você não precisa digitar tudo, por exemplo, basta digitar `html` e selecionar a opção `html:5` e você terá um modelo básico de página HTML para continuar o seu site:

![HTML:5](imagens/Xiii-15.png)

Esse é o modelo básico que vamos modificar de acordo com as nossas necessidades:

![Modelo básico](imagens/Xiii-16.png)

Modifique o idioma para português do Brasil:

![pt_BR](imagens/Xiii-17.png)

Coloque um título interessante para a sua nova página:

![Título](imagens/Xiii-18.png)

Coloque o texto que você quiser:

![Texto](imagens/Xiii-19.png)

Note que ainda não salvamos o arquivo HTML que está sendo editado.
Podemos perceber isso pela bolinha preta ao lado direito do nome do arquivo:

![Não está salvo](imagens/Xiii-20.png)

Para salvar o arquivo, podemos usar a opção `Arquivo` seguido de `Salvar`, ou mais rapidamente, apenas digitar `Ctrl+S`.
Depois disso a bolinha preta vai desaparecer mostrando que o arquivo está atualizado.
Se você modificar novamente o arquivo depois disso, a bolinha preta vai aparecer novamente.
Basta salvar o arquivo novamente e pronto:

![Salvo](imagens/Xiii-21.png)

## Passo 5

Vendo o resultado (até agora)

Abra o seu `Explorador de Arquivos` do seu Windows.
Navegue até a pasta onde você salvou o seu site.
O meu eu salvei em `Meus Documentos`, então dê um duplo clique nessa pasta:

![Meus Documentos](imagens/Xiii-22.png)

Dentro de `Meus Documentos`, eu criei a pasta `meus-sites`, novamente, dê um duplo clique nessa pasta:

![meus-sites](imagens/Xiii-23.png)

E, finalmente, encontramos a pasta onde eu salvei o meu novo site.
A pasta se chama `exemplo`.
Dê um duplo clique nessa pasta:

![exemplo](imagens/Xiii-24.png)

Na pasta `exemplo` você vai encontrar o arquivo `index.html` que nós acabamos de criar.
Para ver o seu conteúdo no seu navegador, você pode arrastar o arquivo até a janela do navegador, ou simplesmente dar um duplo clique no ícone do arquivo:

![index.html](imagens/Xiii-25.png)

Pronto!
O resultado é exibido no navegador!

![Navegador](imagens/Xiii-26.png)

Se você alterar o conteúdo do arquivo `index.html` (isso vale para qualquer arquivo Web), você tem que lembrar de salvar o arquivo novamente e apertar a tecla F5 no navegador.
